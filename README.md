# CSI_SE

> GitHub Pages, Vercel, Netlify 등 무료 웹 호스팅에 배포 가능한 프로젝트입니다.

## 🚀 로컬에서 보기

- **정적 HTML**: `index.html`을 브라우저로 열기 또는 `npx serve .`
- **Node/React 등**: `npm install` 후 `npm run dev` 또는 `npm run build` 후 `npm run preview`

## 📦 GitHub에 올리기

```bash
cd D:\whanin_rpa\CSI_SE

git init
git add .
git commit -m "Initial commit"

# GitHub에서 새 저장소(예: CSI_SE) 생성 후
git remote add origin https://github.com/your-username/CSI_SE.git
git branch -M main
git push -u origin main
```

## 🌐 무료 웹 호스팅

| 플랫폼 | 방법 |
|--------|------|
| **GitHub Pages** | 저장소 → Settings → Pages → Source: GitHub Actions 또는 main |
| **Vercel** | vercel.com → Import Git Repository → CSI_SE 선택 |
| **Netlify** | netlify.com → Add new site → Import from Git → CSI_SE 선택 |
