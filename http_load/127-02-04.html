[127-02-04] 건설현장의 근로자 중에 주의력있는 근로자와 부주의한 현상을 보이는
근로자가 있다. 부주의한 근로자의 사고를 예방할 수 있는 안전대책에 대하여 설명하시오.
부주의 개요
주의 / 부주의 정의 / 예방 관점
주의: 의식작용이 있는 사물에 집중하는 상태 부주의: 주의 저하·산만 상태재해예방 위해 부주의를 유발한 요인 파악 및 대책 수립 필요
부주의 현상 특징
재해 발생 조건 / 정신상태 / 원인 조건 / 유사현상 구분
재해는 불안전한 상태와 불안전한 행동의 중첩 시 다발부주의는 실패 동작 시의 정신상태로 나타남부주의 유발 조건 제거 필요착각·오인·지능부족 등 유사 실패는 부주의로 단정 불가
의식수준과 부주의
의식 단계 특성 / 작업 적정 단계
단계 0~IV로 의식수준·주의상태·신뢰도 변화 (수면→졸음→일상→적극활동→과긴장)안전 작업은 일상생활 단계에서 수행이 바람직최적 단계(일상~적극활동)가 적절히 반복되도록 공정·작업배치 설계 필요
부주의 원인
외적 요인 / 내적 요인
외적 요인: 작업환경조건 불량, 작업순서 부적당, 작업강도(작업량·시간·속도), 기상조건(온도·습도 등)
외적 요인 / 내적 요인
내적 요인: 소질적 요인, 의식 우회, 경험부족·미숙련, 피로·스트레스·질병, 정서 불안정
안전 대책
교육·행동 / 건강·업무 / 정신적 대책 / 기능·작업 대책 / 설비·환경 대책
지식·기능·태도 습득 안전교육 철저 착각·오인 유발 요소 제거, 즉응 동작 기준 설정·체득동작 장애요인 제거, 심신 건강 정상 유지능력 초과 업무 부여 금지, 불건전 상태 작업 금지정신적 대책: 주의집중 혼란 방지, 안전의식 제고, 스트레스 해소, 작업의욕 고취기능·작업 대책: 표준동작 습관화, 작업조건 개선·적응력 향상, 안전작업방법 습득, 적성배치설비·환경 대책: 표준작업제도 도입, 설비·작업환경 안전화, 긴급 시 안전작업대책 수립
결론
원인별 예방 / 관리 역할
부주의 예방은 원인별 대책 수립이 핵심안전관리자 모범 수행, 부주의 연구·교육 강화, 적정 작업량 배정, 근로자 의욕 고취가 효과적
