기본 기능

외부·내부 자극·정보 입수·인지
정보보관
인간 학습 기반 기억 축적, 기계 기록·매체 기반 저장
정보처리·의사결정
인간 결심 수반 처리, 기계 사전 프로그램 범위 처리
행동기능
의사결정 결과 기반 조작행위, 물리적 조종·통신행위
인간·기계 상대적 장단점
인간 특성
융통성 우수, 일관성 있는 작업 추진 곤란
기계 특성
융통성 부족, 일관성 있는 작업 추진 가능
인간·기계 우수 기능 비교
감지기능
인간: 예기치 못한 사실 감지 / 기계: 드물게 발생하는 사실 감지
정보저장기능
인간: 장시간 보관 가능 / 기계: 신속·대량 보관
정보처리기능
인간: 다양한 문제 해결 / 기계: 정량적 정보처리
행동기능
인간: 과부하 시 중요 업무 전념 / 기계: 과부하 상태에서도 효율적 작동
결론·시사점
사고예방·생산성
인간·기계 특성 이해 기반 상호 보완, 불안전 행동·상태 접촉 예방
설계·환경 적합
기계·환경 인간 적합화 통한 피로 저감, 휴먼에러 감소, 안전·능률 동시 확보
