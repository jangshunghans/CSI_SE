[138-01-08] 작업환경 개선을 위한 ECRS 원칙
개요
정의
작업공정 및 작업환경을 체계적으로 개선하기 위한 4단계 원칙 : Eliminate(제거) → Combine(결합) → Rearrange(재배열) → Simplify(단순화)
E (Eliminate)
제거
불필요한 작업, 동작, 위험요인 자체를 제거 예) 고소작업 → 지상에서 조립 후 설치
C (Combine)
결합
유사 작업을 통합하여 이동·대기 시간 단축 예) 검사·운반 작업 동시 수행
R (Rearrange)
재배열
작업 순서·배치를 최적화하여 효율·안전성 향상 예) 위험작업을 안전한 구역으로 재배치
S (Simplify)
단순화
작업 방법·도구를 단순하게 개선하여 실수 감소 예) 표준작업절차서 수립, 치공구 개선
적용효과
재해예방
근원적 위험 제거(E)로 사고 가능성 최소화
적용효과
생산성
불필요한 동작 제거·단순화로 작업 효율 향상
적용효과
근골격계
작업 재배열·단순화로 부적절한 자세 감소
건설현장 적용예
갱폼(Gang Form)
지상 조립 후 양중 설치 → 고소 조립 작업 제거(E)
건설현장 적용예
자재 운반
운반 경로 최적화 → 이동거리 단축(R)
