[131-03-01] 인간의 의식수준과 부주의 행동관계에 대하여 설명하고,
휴먼 에러의 심리적 과오에 대하여 설명하시오.
개요
긴장수준과 의식수준
긴장수준이 인간의 의식·기능 수준을 나타내는 지표, 긴장 저하 시 기능 저하·불쾌증상·사고경향 증가
긴장수준과 사고경향
긴장수준 저하 시 부주의 행동·휴먼에러 발생 가능성 증대
의식수준과 부주의 행동
Phase I 특징
심한 피로, “몹시 귀찮음” 감정 선행, 단순·조잡한 부주의 행동 발생 단계
Phase I
Phase I 인지·확인 에러
신호·변화 무관심·간과, 귀찮음으로 점검·확인 생략
Phase I 기억·판단 에러
지시·연락사항 망각, 기본적인 기억 누락
Phase I 동작·조작 에러
눈앞 상황에 안이한 개입(장면행동), 감정적·난폭한 취급, 성급한 작업 마감
Phase II
Phase II 특징
예기치 않은 사태에 주의 빼앗김, 지레짐작·부적절한 판단 증가 단계
Phase II 인지·확인 에러
돌발 상황에 주의 집중, 확인 없이 잘못 인지, 오해·지레짐작
Phase II 기억·판단 에러
위험 인식 일시 망각, 점검 불필요 과신(예측과잉), 과거 성공 경험에 따른 안이한 판단, 연락 생략, 절차 누락·왜곡, 끼어든 일로 인한 절차 혼선
Phase II 동작·조작 에러
잠시 대기 대신 다른 작업 착수로 타이밍 상실, 습관적 돌출 행동, 반사적 손 내밀기, 지름길·단락 반응
Phase III
Phase III 특징
주의력이 가장 잘 작동, 주의 범위 넓으나 시간·상황 압박 시 오판 발생 단계
Phase III 인지·확인 에러
과도한 사고집중으로 처리해야 할 다른 일을 놓치는 인지 누락
Phase III 기억·판단 에러
고장·수리 등에 몰입하여 시간 경과·상황 변화 인지 지연, 타이밍 상실
Phase III 동작·조작 에러
시간 압박·상황 급변으로 즉시 판단 요구 시 행동 선택 오류
Phase IV
Phase IV 특징
눈앞 돌발 상황에 놀람·당황이 지배, 패닉에 가까운 부주의 행동 단계
Phase IV 인지·확인 에러
돌발사태에 놀라 일점 집중, 다른 정보·신호 무시
Phase IV 기억·판단 에러
과도한 긴장·당황으로 판단 혼란·불능, 분노·공포로 냉정한 판단 불가
Phase IV 동작·조작 에러
패닉 상태에서 무목적 동작 반복, 과도한 긴장·당황에 따른 실수 빈발
휴먼에러 심리적 과오
생략의 에러(Omission)
필요한 작업·단계의 미수행, 절차 일부 또는 전부 빠뜨리는 과오
시간의 에러(Timing)
주어진 시간 내 동작 미수행, 과도하게 빠르거나 느린 수행으로 인한 과오
실행의 에러(Commision)
작업·단계는 수행했으나 방법·내용이 잘못된 실행 오류
순서의 에러(Sequential)
작업·단계의 수행 순서를 잘못 적용한 과오
과잉행동 에러(Extraneous act)
불필요·금지된 작업행동을 추가로 수행한 과오
결론
긴장수준 저하와 휴먼에러 관계
긴장수준 변화·저하 시 인간과오 발생이 쉬운 특성, 부주의 행동 증가에 따른 사고 다발 경향
예방대책 방향
지적확인 활성화 등 사업장 특성에 맞는 다각적 안전대책 수립·시행 필요, 긴장 상태 관리·휴먼에러 심리적 과오 감소를 통한 안전사고 예방
