[125-01-13] 절토 사면의 계측항목과 계측기기 종류
개요
절토사면 계측 목적
시공 안정성 확보, 조기 자료 수집 → 설계·시공에 반영하여 경제적·안전 시공 유도
계측항목·기기
지중경사계/ 다단식경사계
설치: 사면 소단 / 항목·목적: 지중 측방이동량 확인, 수평변위 측정 → 사면 안정성 판단
지하수위계
설치: 사면 소단 / 항목·목적: 사면 지하수위 측정
와이어 변위계(신축계)
설치: 상단–소단, 상단–하단, 상단 / 항목·목적: 지표면 변위량 측정
토압계
설치: 보강구조물 배면지반 / 항목·목적: 보강구조물에 작용하는 토압 변화 측정
철근계
설치: 보강구조물 / 항목·목적: 보강 부재 응력 변화 측정
거치형 경사계
설치: 사면·구조물 위험사면 부위 / 항목·목적: 위험사면 및 구조물 경사 측정
계측빈도·기간
깎기 구간 (지중경사계 ·지하수위계 ·신축계)
깎기 직후~2개월: 주 2회 이상 / 1~3개월: 주 1회 / 3개월 이후: 월 1회
안전관리
이상 시 조치
계측성과 분석·육안관찰 결과 활동파괴 가능성 예측 시 발주자 보고, 신속히 응급대책 수립·시행
