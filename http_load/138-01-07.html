[138-01-07] TBM(Tool Box Meeting)
개요
정의
작업 시작 전 현장 작업자들이 모여 그날의 작업내용·위험요인·안전대책을 공유하는 5~10분간의 소집단 안전미팅
목적
재해예방
당일 작업의 유해위험요인을 사전에 확인하고 대책을 공유하여 재해를 예방
목적
안전의식 제고
반복적인 TBM을 통해 근로자의 안전의식과 참여도를 높임
실시방법
시간
작업 시작 전 5~10분 이내
장소
작업 현장 또는 인접한 안전한 장소
진행자
작업반장 또는 안전담당자
내용
① 전일 안전사항 공유 →
② 당일 작업내용 설명 →
③ 위험요인 및 대책 공유 →
④ 보호구 착용 확인 →
⑤ 건강상태 확인
기록
TBM 실시 내용을 기록지에 기재하고 참석자 서명
효과
즉각성
당일 작업에 맞는 실시간 안전정보 전달
효과
참여도
소집단 활동으로 근로자 자발적 참여 유도
효과
법적근거
산업안전보건법상 작업 시작 전 안전점검 의무 이행 수단으로 활용
