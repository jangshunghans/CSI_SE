[134-01-05] 안전설계기법의 종류와 Fool Proof의 중요 기구
안전설계 기법
안전작동 (Fail Safe)
일부 고장 발생 시 안전측 동작, 위험 상태 이행 방지
기능대행 (Back Up System)
주기능 고장 시 대기 기능의 대행 수행
실수방지 (Fool Proof)
오조작 어려운 구조, 오조작 발생 시 안전 확보
기능회복 (Fail Soft)
일부 고장 시 성능 저하 허용, 전체 정지 회피
다중계화 (병렬작동)
동일·동종 기능 다중 설치, 교체·선택·병렬 운용
자동진단·회복
고장 검출, 신속 기능 회복 유도
안전율 적용
안전여유 반영 설계, 허용하중·강도 확보
위험부위 고장 감소
위험부 고장 빈도 저감 설계
실수방지 중요기구
가드 (Guard)
위험부 접근·접촉 방지 덮개·울타리·울 등
조작기구
양수조작기구 중심, 양손 동시 조작 유도, 손 위험영역 격리
록 기구 (Lock)
조건 충족 후 다음 동작 허용 기구(수동·자동)
트립 기구 (Trip)
브레이크 연동 급정지 기구
오버런 기구 (Over Run)
완전 정지 전 가드 개방 방지 기구
에러 방지 개선
인간 오류 억제 구조·기계적 개선
