양중기 개요
정의 / 점검 필요성
동력을 사용해 중량물을 매달아 상하·좌우로 운반하는 기계중량물 취급·운반 장치에는 방호장치 설치 필요 → 작업 시작 전 안전점검 필수
방호장치
과부하방지장치
정격하중 초과 시 자동 상승중지 + 경보음·경보 등 작동사용자 임의조정 방지(외함 봉인)· 안전인증 제품 적용 전자식: 스트레인게이지 등 전자감응으로 과부하 감시(크레인·리프트·곤돌라·승강기) 전기식: 권상모터 전류변화로 과부하 감시 (크레인)기계식: 스프링·방진고무 처짐 등으로 리밋스위치 동작 감시(크레인·리프트·곤돌라·승강기)
권과방지장치
권상·기복 와이어로프 권과 방지권과 시 인양금구가 시브·지브에 충돌해 파손·낙하하는 사고 방지요구 성능: 자동 동력차단·제동, 상부/하부 간격 25cm 이상 (직동식 5cm 이상) 확보 종류: 나사식·캠식·중추식·레벨식 리밋스위치
비상정지장치
긴급 시 모든 전원 차단해 급정지 누름 버튼은 적색·돌출형·수동복귀형
제동장치
화물·지브 강하 제동 장치(권상·기복부 설치) 전자브레이크, 전동유압 압상브레이크, 밴드브레이크, 원판브레이크
기타 방호장치
훅해지장치: 훅에서 와이어로프 이탈 방지 충돌방지장치: 동일 주행로 이동식 설비 간 충돌 방지레일정지기구, 안전밸브 등
정기검사
주기 / 항목 / 기록
6개월마다 1회 이상 정기검사항목: 과부하·권과 등 안전장치 이상, 브레이크·클러치 이상, 와이어로프·달기체인 이상, 훅 등 달기기구 손상, 배선·집진·배전반·개폐기·컨트롤러 이상자체검사 결과 2년간 기록 보존
작업 시작 전 점검
크레인 작업
권과방지장치, 브레이크, 클러치·운전장치 기능 확인 주행로 상측·트롤리 횡행 레일 상태 확인 와이어로프 통과부 상태 확인
이동식 크레인 작업
권과방지·경보장치 기능 확인 브레이크·클러치·조정장치 기능 확인 와이어로프 통과부 및 작업장 지반상태 확인
고리걸이 작업
양중기·와이어로프 등 이상 유무 확인
슬링 작업
훅 부착 슬링·와이어슬링 걸림 상태 확인 슬링·와이어슬링 상태를 작업 전·작업 중 수시 점검
결론
관리 원칙
