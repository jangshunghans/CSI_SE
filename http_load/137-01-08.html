[137-01-08] 액상화의 정의 및 간편 예측 방법
액상화
정의
포화 지반 지진 작용, 강성 상실, 액체 유사 거동
발생 개념
간극수압 상승, 유효응력 감소, 외력 저항 상실
액상화 간편 예측
평가 지표
반복저항응력비(CRR)와 반복전단응력비(CSR) 비교
안전율
FS = CRR / CSR
판정 기준-안정
FS > 1.5, 안정
추가검토
1.0 ≤ FS < 1.5, 상세 예측방법 적용 대상
불안정
FS < 1.0, 불안정(액상화 가능성 높음)
상세 예측 판정
FS > 1.0 안정, FS < 1.0 불안정
반복전단응력비(CSR)
산정 방법
지반응답해석 기반 산정
산정 식
CSR = 0.65 × (τmax)d / σ′v
변수 의미
(Tmax)d 지반응답해석 최대 전단응력, σ′v 해당 깊이 연직 유효응력
액상화 평가 절차
사전 검토
주상도, 입도분포 자료 기반 평가 필요성 검토
내진 I 등급
실내 변형특성 평가시험 결과 활용 지진응답해석, 진동삼축시험 기반 CRR 산정
내진 II 등급
간편예측법 우선 적용, FS < 1.5 시 진동삼축시험 기반 상세 예측법 적용
