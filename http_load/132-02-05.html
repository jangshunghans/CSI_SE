[132-02-05] 인간의 긴장정도(Tension Level)를 표시하는
의식수준 5단계와 의식수준과 부주의 행동의 관계에 대하여 설명하시오.
의식수준 개요
개념
긴장 정도를 뇌파 변화로 나타내는 인간의 의식 상태 수준 개념
긴장수준과 사고
긴장수준 저하 시 기능 저하·불쾌 증상 및 휴먼에러·사고경향 증가
의식수준 5단계
0단계
수면 중 상태, 의식 단절·의식 우회, 신뢰도 0
I단계
졸음 상태, 의식수준 저하, 신뢰도 0.9 이하
II단계
일상생활 수준, 일반적인 각성 상태, 신뢰도 0.99~0.99999
III단계
적극활동 상태, 높은 주의집중·활동성, 장시간(15분 이상) 지속 곤란
IV단계
과긴장 상태, 주의의 일점 집중, 신뢰도 0.9 이하
안전작업 권장단계
작업 안전 확보를 위한 II단계 중심 작업, II~III단계의 반복 배치 필요
Phase I 부주의행동
특징
심한 피로·귀찮음 선행, 동기 저하에 따른 전반적 부주의 증가
인지·확인 에러
신호·변화 무시·간과, 귀찮음으로 점검·확인 생략
기억·판단 에러
지시·연락사항 망각, 해야 할 일 기억 누락
동작·조작 에러
눈앞 상황에 안이하게 손댐, 감정적 난폭 취급, 성급한 작업 마무리
Phase II 부주의행동
특징
예기치 않은 사태에 주의 빼앗김, 지레짐작·오판 증가
인지·확인 에러
돌발 상황에 주의 집중, 확인 없이 잘못 인지, 오해·지레짐작 발생
기억·판단 에러 (위험 인식)
위험 인식에도 순간적 망각, “괜찮다”는 과신, 점검 불필요 판단
기억·판단 에러 (절차 누락)
이전 성공 경험 의존, 상대방도 안다고 생각해 미연락, 작업 완료 착각, 끼어든 일·다음 작업 걱정으로 절차 누락·오절차
동작·조작 에러
잠시 대기를 못하고 다른 일 수행해 타이밍 상실, 습관동작 통제 실패, 반사적 손짓, 지름길·단락 반응
Phase III 부주의행동
특징
주의력·주의범위 최적이지만 시간·상황 압박 시 판단에러 발생
인지·확인 에러
과도한 과제 복잡성으로 사고 몰입, 처리해야 할 일·신호 누락
기억·판단 에러
시간 압박·상황 급변 속 즉시 판단 요구, 고장·수리 작업 몰입으로 상황·시간 경과 인지 실패·타이밍 상실
동작·조작 에러
시간 절약·급변 상황에서 성급한 조작, 판단 미숙에 따른 잘못된 동작
Phase IV 부주의행동
특징
돌발사태·공포·분노에 의한 과긴장·패닉 상태
인지·확인 에러
돌발사태에 놀라 일점 집중, 주변 정보·다른 신호 무시
기억·판단 에러
과도한 긴장·당황으로 판단 혼란·불능, 분노·공포로 냉정한 판단 곤란
동작·조작 에러
패닉 상태에서 무목적·무의미 동작 반복, 흥분으로 인한 연속 실수
결론·시사점
긴장수준과 휴먼에러
긴장수준 변화·저하 시 휴먼에러 발생 증가, 안전성과 직결되는 인간 특성
안전작업 관리 방향
II단계 중심 의식수준 유지, 공정 배치를 통해 II~III단계가 반복되도록 작업 설계 필요
안전관리 적용
피로·과긴장 관리, 작업강도·휴식·교육·환경개선을 통한 휴먼에러 저감
