[138-01-11] 「산업안전보건기준에 관한 규칙」상 이동식 비계를 조립하여 작업하는 경우 준수사항
개요
정의
이동식 비계: 바퀴(캐스터)가 달려 이동 가능한 강관 비계
준수사항
아웃트리거
비계 최하부에 아웃트리거(outrigger)를 설치하거나 전도 방지 조치를 취할 것
바퀴 고정
작업 중 이동을 방지하기 위해 브레이크·쐐기 등으로 바퀴를 고정할 것
작업발판
작업발판의 최대 적재하중은 250 kg을 초과하지 않도록 하고, 발판을 견고히 설치할 것
안전난간
작업발판 주위에 안전난간(상부난간대 90~120 cm, 중간난간대, 발끝막이판) 설치
이동 시 조건
비계 이동 시에는 작업자가 비계 위에 탑승하지 않을 것
이동 경로
비계를 이동할 때 구멍·요철·경사 등 장애물이 없는 평탄한 지면에서 이동할 것
높이 제한
비계 높이는 밑변 최소폭의 4배 이하로 할 것 (일반적으로 최대 높이 제한 준수)
승강설비
작업자 승·하강을 위한 안전한 승강설비(사다리 등)를 설치할 것
재료 상태
비계 사용 전 부재의 변형·부식·균열 여부를 점검할 것
과하중 금지
비계 위에 재료를 적재할 때 최대 적재하중을 초과하지 않을 것
