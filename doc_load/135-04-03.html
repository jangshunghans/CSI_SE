[135-04-03] 건설기계 중 화물의 적재/하차 시 사용되는

지게차의 유해 위험요인 및 예방 대책과 작업단계별 안전점검사항에 대하여 설명하시오.

구분 세부항목 설명
개요 지게차 정의 차체 전면 포크(fork) 활용 화물 적재·하역·운반 운반기계
관리 필요성 적재·운행·하차 단계 유해·위험요인 관리, 예방대책 및 단계별 안전점검 준수
유해·위험 요인 물체 낙하 화물 불안전 적재, 부적당 작업장치 선정, 운전 미숙, 급출발·급정지·급회전
충돌·협착 구조상 시야 불리, 후륜주행 특성에 따른 회전반경 영향
차량 전도 과적재, 급회전, 요철 바닥면 미정비, 화물 대비 소형 차량 사용
예방대책 안전벨트 전도·급출발 시 운전자 차체 외 이탈 방지
후방접근 경보장치 후진 시 물체·근로자 충돌 예방, 후진기어 변속 시·접근 시 경보음
장애물 감지거리 최대 15m 이내 감지 기준
대형 후사경 소형 후사경 대비 확인 시야 확대, 규격 18.5cm × 21cm
룸미러 사각지대 해소, 규격 26cm × 10cm
포크 높낮이 표시기 포크 바닥면 20~30cm 유지 확인 목적 지시기 설치
포크·마스트 시각표시 포크 화살표 지시기, 마스트 안전위치 마킹
안전문 전도 시 운전자 튕김 방지
안전모 전도·낙하물 대비 머리 보호
형광테이프 조명 취약 환경 지게차 위치 가시성 확보
경광등 창고·조도 불량 장소 운행 상태 표시
포크 받침대 수리·정비 시 포크 급강하 방지
소화기 지게차·작업장 화재 초기 대응
작업단계별 안전점검 작업 시작 전 점검 체계 안전점검표 기반 엔진 시동 전·후, 서행 주행 중 점검
작업 시작 전 주요 항목 외관, 바퀴·타이어, 방향지시기, 램프, 백미러, 경보장치, 브레이크, 클러치, 헤드가드
시동 전 레버·기어 기어변속·각 작용 레버 중립(정위치) 확인
시동 전 브레이크 핸드브레이크 체결 상태 확인
시동 후 엔진 상태 저속회전 여부, 회전음·폭발음·배기가스 상태, 이상 유무
작업 종료 후 정비 청소, 오염 심한 경우 물 세척
기록 관리 정해진 항목 점검, 주행일지 기록
윤활·유체 관리 회전부 손질 후 주유, 연료·윤활유·냉각수 충전
정기점검 주기 매월 1회 작동상태 점검
정기점검 주요 항목 동력전달장치, 조종장치, 제동장치, 하역장치, 유압장치, 종합테스트
결론 현장 적용 포인트 유해·위험요인별 대책 준수, 단계별 점검 이행, 사전조사·작업계획서 기반 운행 관리
