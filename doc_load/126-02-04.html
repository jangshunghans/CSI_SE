[126-02-04] 한중콘크리트 시공 시 문제점과 안전관리대책에 대하여 설명하시오.

구분 세부 항목 주요 내용
개요 한중콘크리트 정의 일평균 기온 4℃ 이하 조건에서 타설·양생 중 동결 우려가 있는 경우 사용하는 콘크리트
시공 필요성 저온으로 수화반응·응결경화 지연, 강도 증진 저하, 초기 동결 위험 증가 → 설계·재료·배합·시공관리 대책 필수
문제점 초기동해 콘크리트 -0.5 ~ -2.0℃에서 동결 가능, 동결 시 강도·내구성·수밀성 회복 곤란, 초기동해 발생 콘크리트 제거·재시공 필요, 시간·경제적 손실
수화·응결경화 지연 5℃ 이하 노출 시 응결·경화시간 연장, 강도발현 지연, 조기 하중 구조물은 균열·잔류변형 위험
안전관리 대책 재료 선정·보관 조강 포틀랜드 시멘트 사용 검토, 시멘트 직접가열 금지·냉각 방지 보관, 골재·시멘트 동결빙설 혼입 방지 피복, 골재 가열 시 균등가열·과건조 방지, AE제·AE감수제 사용 원칙, W/B 감소로 동결저항 증대, 혼화수 가열이 골재가열보다 유리
배합 AE제·AE감수제 표준 적용, 단위수량 최소화(필요 워커빌리티 확보 범위)
비비기 (온도대책) 4℃ 이상: 일반 시공 / 4~0℃: 타설 콘크리트 보호 / 0~-3℃: 물 또는 물+골재 가열 / -3℃ 이하: 물+골재 가열, 필요 시 급열양생
비비기 (투입순서) 뜨거운 물 → 굵은골재 → 잔골재 → 시멘트, 시멘트는 믹서 내부온도 40℃ 이하 시 투입
비비기 (직후 온도관리) 기상·운반시간 고려해 치기 온도 관리, 배치별 비빈 직후 온도 변동 최소화
운반·치기 운반 중 온도저하 방지, 운반~치기 시간 단축, 철근·거푸집 빙설 제거 후 타설, 구콘크리트·지반 동결 시 제거 후 신콘크리트 타설, 타설 온도 5~20℃ 유지, 얇은 부재 10℃ 이상·두꺼운 부재 5℃ 이상 확보, 펌프 사용 시 배관 예열·보온
양생 (원칙) 치기 후 장시간 외기 노출 금지, 초기동결 방지 바람막이 설치, 양생 중 콘크리트 온도 5℃ 이상 유지
양생 (방법) 보온양생: 단면 크거나 외기온 비교적 높을 때, 단열재 덮어 수화열 이용 / 급열양생: 단면 얇거나 보온만으로 0℃ 이상 유지 곤란 시, 열풍기·전기양생 등 사용
양생 종료 강도 계속·자주 물로 포화되는 부분: 얇은 15MPa / 보통 12MPa / 두꺼운 10MPa, 보통 노출 상태: 전 부재 5MPa
거푸집·동바리 거푸집은 보온성 좋은 재료 사용, 동바리 기초는 동결지반 융해 침하 방지, 탈형 시 차가운 외기와 급접촉 방지
결론 적용 기준 요지 평균기온 4℃ 이하에서는 응결경화 지연과 동결 위험이 크므로 한중콘크리트 기준에 따라 시공·양생해야 함
