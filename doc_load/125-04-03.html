[125-04-03] 건설현장에서 사용되는 고소작업대(차량탑재형)의 구성요소와

안전작업 절차 및 작업 중 준수사항에 대하여 설명하시오.

구분 세부 항목 주요 내용
개요 차량탑재형 고소작업대 정의 주행 제어장치가 차량 운전석 안에 있는 고소작업대
사용 목적·특성 작업대에 탑승한 작업자를 동력으로 붐대 상승시켜 외벽·유리·간판 설치/보수 등 고소작업 수행하는 장비로 건설현장에 주로 사용
안전작업절차 절차 개요 작업계획 수립부터 작업종료까지 단계별 준수사항·안전성 검토/확인 절차 마련
1단계: 작업계획 수립·검토 일반안전사항 검토, 고소작업대 선정 시 고려사항 반영
2단계: 장비 반입·설치 작업 전 점검사항 이행, 설치 시 준수사항 적용
3단계: 작업 실시 작업 중 안전수칙 준수, 장비 작동 절차에 따라 운전
4단계: 작업 종료 종료 시 안전수칙 준수, 장비 이상 유무 확인
주요 안전점검사항 작업계획서 ·사전조사 작업계획서 작성 여부, 수리·점검 이력관리, 안전검사 실시 여부 확인
운전자격 운전원 면허·자격 적정 여부 확인
목적 외 사용금지 임의개조 금지, 안전장치 해제 사용 금지 여부 확인
작업 전·중 준수사항 기계 이상유무 점검, 유도자·신호수 배치, 탑승근로자 안전대 착용·부착 및 보호구 착용, 유압·조작장치 등 주요 구조부 상태 확인
안전장치 점검 통합 컨트롤러 붐 길이·각도·적재하중·작업반경 제한 기능 정상, 해제 여부 확인
모니터링 장치 붐 길이·각도·하중·반경 표시 정상, 작업반경 초과 시 자동정지 확인
과부하 방지장치 (로드셀) 탑승하중 측정·모니터링 정상, 과부하 시 정지 확인
붐 각도·길이 센서 각도·길이 변경 시 모니터 정상작동 여부 확인
자동안전장치 (인터록) 붐 상승 상태에서 아우트리거 작동 불가 연동 확인
아우트리거 전복방지센서 센서 위치·작동상태 확인(슬라이드박스/잭실린더)
구조부 점검 붐·턴테이블 붐대 균열(기복실린더 로드 연결 용접부)·턴테이블 균열, 체결볼트 풀림 확인
작업대 (탑승함) 고정볼트 풀림·핀 탈락 여부, 난간 휨·찌그러짐·파손상태 확인
아우트리거 잭 발판 고정, 완전확장 여부, 유압호스 누설 확인
인출 와이어/체인 마모·단선·파손 여부, 인출 롤러 마찰부 마모상태 확인
결론 관리 방향 현장 관계자의 관리·감독 미흡으로 장비·주변 근로자 안전 확보가 부족 → 안전작업절차·작업계획서를 쉽게 관리할 수 있는 안전작업 가이드 보급으로 장비 재해 예방 필요
