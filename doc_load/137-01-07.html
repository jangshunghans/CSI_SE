[137-01-07] 침윤선의 정의 및 작도 방법

구분 세부항목 설명
개요 정의 하천 제방·필댐 등 제체 통과 침투수 작용 한계선, 수압 0 유선, 자유수면 개념
적용 의미 제체 유선망 분포 평가 기준, 위치별 수압 크기·분포 검토 기준, 파이핑 안정성 검토 기준
침윤선 작도방법(카사그란데) 기준점 설정 AE = 0.3AG 조건 만족 E점 설정
유점 설정 B점에서 연직선 작성, 수면 교점으로 U점 설정
포물선 구성 C점 초점 설정, D점·E점 통과 포물선 작성
침윤선 연결 A점에서 등수두선 AB와 직각 교차, 기본 포물선 접속, A–J–D–U 곡선 형성
D점 산정 (X+S)² = X² + H² 관계 적용, S = √(X²+H²) − X 산정, CD = 1/2·S 조건으로 D점 결정
중요성 안정성 영향 지하수 거동 평가 핵심 요소, 댐·제방 안정성 좌우 요소
재해 예방 제체 간극수압 관리 기준, 파이핑 방지, 사면 붕괴 예방 기반
