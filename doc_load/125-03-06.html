[125-03-06] 중대재해 발생 시 [산업안전보건법령] 에서 규정하고 있는 사업주의 조치 사항과

고용노동부 장관의 작업중지 조치 기준 및 중대재해 원인조사 내용에 대하여 설명

구분 세부 항목 주요 내용
개요 작업중지 제도 취지 산업안전보건법은 재해 발생 위험 시 사용중지·작업중지 명령을 통해 사업주가 재해를 강제 예방하도록 함
사전 숙지 필요 중대재해 발생 시 사업주 조치, 고용노동부장관 작업중지 기준, 원인조사 내용을 사전 숙지해야 함
사업주의 조치사항 중대재해 발생 시 조치 (법 제54조) 중대재해 발생 즉시 해당 작업 중지, 근로자 대피, 필요한 안전·보건 조치 시행
중대재해 보고 (시행규칙 제67조) 중대재해 인지 시 지체 없이 관할 지방고용노동관서장에게 전화·팩스 등으로 보고
보고 내용 발생 개요·피해상황, 조치 및 전망, 기타 중요 사항
고용노동부장관 작업중지 조치기준 시정조치 ·사용중지 (법 제53조) 안전·보건 조치 미이행으로 현저한 위험 우려 시 기계·설비 사용중지/대체/제거/개선 등 시정조치 명령 가능
시정조치 게시 시정조치 완료 시까지 명령사항을 근로자가 쉽게 볼 수 있는 장소에 게시
작업중지 명령 시정조치 미이행으로 위험 해소 안 되거나 위험이 현저히 높아질 우려 시 작업 전부 또는 일부 중지 명령
해제 절차 사업주가 시정조치 완료 후 사용중지·작업중지 해제 요청 가능, 장관은 완료 확인 시 해제
중대재해 발생 시 작업중지 작업중지 명령 대상 (법 제55조) 중대재해 발생 후 동일·유사 작업에서 재발 급박 위험이 있으면 해당 작업 또는 동일 작업 중지 명령
사업장 전체 중지 가능 토사·구축물 붕괴, 화재·폭발, 유해물질 누출 등으로 재해 확산 우려 ㅛ; 불가피 시 사업장 작업 중지 가능
해제 심의 사업주 해제 요청 시 전문가 등 심의위원회 심의를 거쳐 작업중지 해제
결론 운영 개선 방향 작업중지 제도의 공정·투명 운영을 위해 근로자·사업주·장관 운영 실태를 주기 점검하고 문제점 개선·보완 필요
