[126-01-09] 터널 제어발파

구분 세부 항목 주요 내용
개념 터널 제어발파 정의 천공방법·장약방법·발파시간 등을 조절하여 여굴 방지, 모암 손상 최소화, 저소음·저진동으로 굴착면 품질을 높이는 발파공법
특징 발파면 매끈성 발파단면 매끈, 굴착면 평활, 여굴 감소
낙석 위험 저감 불규칙 파쇄 감소로 낙석 위험 축소
모암 손상 저감 발파 에너지 집중 제어로 모암 손상 최소화
복공 콘크리트 절감 여굴 감소로 복공 Con’c 물량 절약
버력량 감소 부석·버력량 감소로 운반·처리 효율 향상
종류 라인 드릴링 (Line Drilling) 설계 윤곽선을 따라 무장약 또는 소장약 천공열 형성 후 주발파 수행
라인 드릴링 장약·순서 자유면 → 100% → 50% → 0% (윤곽부는 무장약/최소장약)
라인 드릴링 특징 매끈한 발파면 확보, 정교한 윤곽 형성
프리 스플리팅 (Pre-splitting) 본발파 전에 윤곽선에 선행 균열면(절리면)을 만들어 여굴 억제
프리 스플리팅 장약·순서 50% → 100% → 50% → 자유면
프리스플리팅 특징 여굴 최소화 효과 큼, 모암 보호 우수
쿠션 블라스팅 (Cushion Blasting) 윤곽부에 분산장약(Decking)으로 에너지 완화 후 주발파
쿠션 블라스팅 장약·순서 자유면 → 100% → 100% → 분산장약
쿠션 블라스팅 특징 장약량 최소화, 윤곽부 과다파쇄 방지
