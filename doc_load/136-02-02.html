[136-02-02] 자극-반응(Stimulus-Response)이론에 대하여 설명하시오.

구분 세부항목 설명
개요 자극-반응 이론 개념 자극(stimulus)과 반응(response) 연결 중심 학습 설명, 동일 자극-동일 반응 가정
손다이크 관점 자발적·자기주도적 행동 중심, 자극-자발적 반응 연결 관심
주요 개념 자극 (stimulus) 외부 입력·사건, 시각·청각·촉각 등 감각자극, 행동 유발 요인
반응 (response) 자극 이후 행동·신체 반응·정서 반응
주요 이론 왓슨 자극-반응 이론 관찰 가능한 행동 분석 중시, 조건화된 반응 형성에서 자극 역할 강조
손다이크 시행착오설 (Trial and Error Theory) 시행-오류 반복 통한 자극-반응 결합 형성
연습·반복의 법칙 반복 연습 기반 반응 강화, 망각 방지
효과의 법칙 쾌감·만족 결과 기반 반응 강화
준비성의 법칙 기초 능력·준비 상태 기반 학습 효율 증대
분석의 법칙 예시 반복 후 공통 원리 파악 중심 학습
파블로프 조건반사설 조건 형성 기반 새로운 행동 성립 설명, 무조건 자극-조건 자극 결합
시간의 원리 시간적 근접성 기반 조건화 촉진
강도의 원리 반응 강도 점진적 강화 경향
일관성의 원리 동일 자극 지속 결합 기반 조건 성립
계속성의 원리 일정 간격 다회 반복 기반 조건 유지
학습 유형 고전적 조건화 (Classical Conditioning) 무조건 자극-조건 자극 결합, 조건 자극 단독 반응 유발 과정
조작적 조건화 (Operant Conditioning) 보상·처벌 기반 행동 빈도 증감, 강화 중심 행동 조형
현장 적용방안 감각적 자극 강화 소리·색상·점멸 등 시청각 자극 활용, 체험형 교육 설계
반복 교육 위험상황 반복 훈련 기반 조건화
피드백 시스템 안전행동 결과 기반 보상·제재 운영
행동기록·분석 자극-반응 패턴 기록, 개선요인 도출
결론 교육적 시사점 행동 관찰·측정 중심, 목표 명확화, 참여·실습 강화, 피드백 중요성
