[135-01-11] Soil Nailing과 Earth Anchor에 대한 설명과 비교

구분 세부항목 설명
개요 공법 개념 비교 Earth Anchor 공법 긴장 지반앵커 적용, Soil Nailing 공법 무긴장 보강재 적용
공법 선정 고려 시공성, 공사기간, 공사비 중심 적정 공법 선정
Earth Anchor 공법 적용 개념 긴장력 도입 앵커 기반 사면·흙막이 변형 억제
시공 방식 Top-Down 방식 적용
시공 흐름 엄지말뚝 시공 → 굴착 및 띠장 설치 → Earth Anchor 설치 → Rock Bolt 시공
장비 특성 대규모 장비(T-4) 활용
장점 시공경험 축적
장점 인장력 확보 시 지반변형 억제 효과 큼
단점 대규모 장비 투입에 따른 시공성 저하
단점 다공종 투입에 따른 품질관리 난이도 증가
단점 터파기 완료 후 강재 회수 곤란
단점 앵커 분담력 상대적 증가, 국부 문제 시 전체 안정 영향 확대
단점 공기 장기화, 공사비 증가
Soil Nailing 공법 적용 개념 무긴장 Nail 다수 배치 기반 사면·굴착면 보강
시공 방식 Top-Down 방식 적용
시공 흐름 굴착 → Shotcrete → 천공 및 Nail 설치 → 반복 공정
장비 특성 소형 장비(Crawler Drill) 중심
장점 시공 중 토질조건 변화 대응 용이
장점 시공 용이성 우수
장점 공기 단축, 공사비 절감
장점 Nail 간격 소, 국부 품질 문제의 전체 안정 영향 감소
단점 정밀 시공관리 요구
