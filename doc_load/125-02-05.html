[125-02-05] 재해손실 비용 산정 시 고려사항 및 Heinrich 방식과 Simonds 방식을 비교 설명

구분 세부 항목 주요 내용
개요 재해손실비용 정의 업무상 재해(인적상해 수반)로 인해 발생하는 직·간접 손실비용. 재해가 없었다면 지출하지 않을 비용
재해손실비용 산정 고려사항 (충족조건) 신뢰성 사회 다수가 신뢰할 만한 확률·근거가 있을 것
간편성 안전관리자가 일상업무로 처리 가능한 간단한 방법일 것
보편성 대규모·중소규모 사업장 모두 일률 적용 가능할 것
집계가능성 산업별·전국적 집계(추계) 가능할 것
하인리히 vs 시몬즈 비교 Cost 분류 하인리히: 직접비·간접비 / 시몬즈: 보험 Cost·비보험 Cost
직접비 (보험 Cost) 하인리히 직접비=산재보상비 / 시몬즈 보험 Cost=보험금 총액+보험회사 경비·이익
보험료-보상차액 하인리히: 미가산 / 시몬즈: 보험 Cost에 가산
간접비·비보험 Cost 개념 유사하나 구성항목 차이 존재
총 재해비용 산정 하인리히: 직·간접 1:4(계산 용이, 평균치법) / 시몬즈: 1:4 부정, 새로운 평균치법
장단점 하인리히: 산업별 비율 변동 (중공업 1:10~20, 경공업 1:1~2)로 적용 제한 / 시몬즈: 전체산업 적용 가능하나 계산 곤란, 사업장별 산정 가능
국내 적용 적용 방식 우리나라는 하인리히 방식 적용
산정식 재해손실비용 = 직접손실액 + 간접손실액 = 직접손실액 × 5(간접=직접×4)
주요 차이점 분류기준 시몬즈 =보험/비보험, 하인리히=직접/간접
보험차액 시몬즈는 보험 Cost 가산, 하인리히는 미가산
간접·비보험 범위 유사 개념이나 세부 항목 구성 차이
비율관점 시몬즈는 하인리히 1:4 전면 부정
